How it works

    ğŸ² Is the wheel truly random?

    Each spin uses a random starting angle combined with a random spin distance. The values are generated in JavaScript using a pseudo-random number generator built into the browser.

    For most classroom or party use-cases this is more than enough to feel fair and unpredictable.
